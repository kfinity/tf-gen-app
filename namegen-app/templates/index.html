Given a sequence of characters, the LSTM tries to predict the next character.

Clearly some more tuning is needed, since it hasn't really figured out when to stop! Output is stopped at 12 characters.

Also, the results for any sequence of characters are currently deterministic, so the same sequence of letters will always return the same result. In development, it is easy enough to add stochastic behavior by sampling from the logit distribution, but this small cloud VM doesn't have the memory to handle the TensorFlow libraries required.
